# ФССП Жалобы — Chrome-расширение

[![CI](https://github.com/alexeymasalykin/fssp-complaints-extension/actions/workflows/ci.yml/badge.svg)](https://github.com/alexeymasalykin/fssp-complaints-extension/actions/workflows/ci.yml)

Chrome-расширение для автоматизации массовой подачи обращений (жалоб) в ФССП через сайт [fssp.gov.ru](https://fssp.gov.ru).

Загружает данные из Excel-файла и последовательно заполняет форму обращения на сайте ФССП, включая каскадные выпадающие списки (vue-select).

## Возможности

- Парсинг Excel-файлов (.xlsx) с данными обращений
- Автоматическое заполнение всех полей формы (текстовые, textarea, vue-select)
- Работа с каскадными зависимостями (регион -> муниципалитет, вид обращения -> тема)
- Умный подбор опций в выпадающих списках (exact / partial / stem match для русских склонений)
- Автоматическая генерация PDF из текста обращения и прикрепление к форме
- Прогресс-бар и навигация по списку обращений
- Все данные хранятся локально в браузере (chrome.storage)

## Стек

- Chrome Extension Manifest V3
- TypeScript (strict mode)
- Vite + [@crxjs/vite-plugin](https://github.com/nicedoc/crxjs)
- SheetJS (xlsx) — парсинг Excel
- jsPDF — генерация PDF
- Vitest — тестирование

## Архитектура

```
┌──────────┐     chrome.runtime      ┌─────────────────┐     chrome.tabs      ┌───────────────┐
│  Popup   │ ◄──────────────────────► │ Service Worker  │ ◄──────────────────► │ Content Script│
│ popup.ts │    sendMessage / port    │ service-worker  │    sendMessage       │   fssp.ts     │
│          │                          │                 │                      │               │
│ - UI     │                          │ - Queue mgmt    │                      │ - DOM fill    │
│ - Excel  │                          │ - State machine │                      │ - vue-select  │
│   parse  │                          │ - Navigation    │                      │ - PDF attach  │
└──────────┘                          └─────────────────┘                      └───────────────┘
                                             │
                                      chrome.storage.local
                                      (complaints, results, settings)
```

**Popup** — интерфейс расширения. Загружает Excel, показывает превью, управляет заполнением.

**Service Worker** — координатор. Хранит очередь обращений, state machine (idle → ready → filling → completed), навигация между обращениями.

**Content Script** — работает на странице fssp.gov.ru. Заполняет текстовые поля через `document.execCommand('insertText')`, взаимодействует с vue-select компонентами через mousedown-события.

## Установка и сборка

```bash
cd extension
npm install
npm run build
```

Загрузка в Chrome:
1. Откройте `chrome://extensions`
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку `extension/dist/`

## Использование

1. Подготовьте Excel-файл с колонками (см. таблицу ниже)
2. Откройте popup расширения и загрузите файл
3. Просмотрите превью и нажмите **Начать заполнение**
4. Расширение откроет форму на fssp.gov.ru и заполнит поля
5. Проверьте данные, введите капчу, нажмите «Подать обращение»
6. Нажмите **Отправлено — Следующая** для перехода к следующей жалобе
