# Установка РКЛ Check

## Требования

- Google Chrome (версия 116 и выше)
- Подтверждённая учётная запись на Госуслугах

## Шаг 1. Распаковка

1. Скачайте файл `rkl-check-extension.zip`
2. Распакуйте архив в любую папку, например `C:\rkl-check` или `~/rkl-check`

> Не удаляйте эту папку — Chrome использует её для работы расширения.

## Шаг 2. Установка в Chrome

1. Откройте Chrome и перейдите по адресу: `chrome://extensions`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку, в которую распаковали архив (где лежит файл `manifest.json`)
5. Расширение появится в списке и на панели расширений Chrome (значок щита)

## Шаг 3. Активация лицензии

1. Нажмите на иконку расширения на панели Chrome (значок щита)
2. Откроется окно активации лицензии
3. Введите полученный лицензионный ключ в формате `RKL-XXXX-XXXX-XXXX`
4. Нажмите **Активировать**
5. После успешной активации откроется главный экран расширения

## Шаг 4. Использование

1. Подготовьте Excel-файл (.xlsx) со списком сотрудников. Обязательные колонки:
   - **Номер** (номер документа)
   - **Дата выдачи** (дата выдачи документа, формат ДД.ММ.ГГГГ)
   - **Дата рождения** (формат ДД.ММ.ГГГГ)
   - Серия и ФИО — необязательные (ФИО используется только для отчёта)

2. Нажмите **Открыть Госуслуги** — откроется страница проверки РКЛ
3. Авторизуйтесь на Госуслугах (если ещё не вошли)
4. Вернитесь в расширение, перетащите Excel-файл в область загрузки
5. Проверьте предпросмотр данных и нажмите **Начать проверку**
6. Дождитесь завершения — результаты появятся в расширении
7. Нажмите **Скачать отчёт (Excel)** или **Открыть полные результаты**

## Обновление расширения

1. Скачайте новый архив `rkl-check-extension.zip`
2. Распакуйте в ту же папку (с заменой файлов)
3. Перейдите в `chrome://extensions`
4. Нажмите кнопку обновления (круглая стрелка) на карточке расширения РКЛ Check

Лицензия сохраняется при обновлении — повторная активация не требуется.

## Настройки

Нажмите на шестерёнку в расширении:

- **Задержка между проверками** — рекомендуется 10 сек (при меньшем значении Госуслуги могут ограничить частоту запросов)
- **Авто-скачивание отчёта** — автоматически скачивать Excel при завершении проверки
- **Уведомления** — Chrome-уведомление при завершении

## Безопасность

Данные сотрудников (ФИО, номера документов, даты рождения) **не покидают ваш браузер**. Сервер лицензий получает только лицензионный ключ и количество проверок.
