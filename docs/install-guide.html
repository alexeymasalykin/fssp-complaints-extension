Требования Распаковка Установка Активация Использование Обновление Настройки Безопасность
Требования
Google Chrome
Версия 116 и выше
Госуслуги
Подтверждённая учётная запись
1
Распаковка
1
Скачайте файл rkl-check-extension.zip
2
Распакуйте архив в любую папку, например C:\rkl-check или ~/rkl-check
Не удаляйте эту папку — Chrome использует её для работы расширения.
Содержимое распакованной папки: manifest.json, assets/, src/
2
Установка в Chrome
1
Откройте Chrome и перейдите по адресу: chrome://extensions
2
Включите Режим разработчика (переключатель в правом верхнем углу)
3
Нажмите Загрузить распакованное расширение
4
Выберите папку, в которую распаковали архив (где лежит файл manifest.json)
5
Расширение появится в списке и на панели расширений Chrome (значок щита)
Страница chrome://extensions — режим разработчика включён, РКЛ Check установлен
Для удобного доступа закрепите расширение: нажмите иконку пазла на панели Chrome, затем кнопку закрепления рядом с РКЛ Check.
3
Активация лицензии
1
Нажмите на иконку расширения на панели Chrome (значок щита)
2
Откроется окно активации лицензии
3
Введите полученный лицензионный ключ: RKL-XXXX-XXXX-XXXX
4
Нажмите Активировать
5
После успешной активации откроется главный экран расширения
Окно активации — введите ключ и нажмите «Активировать»
4
Использование

Подготовьте Excel-файл (.xlsx) со списком сотрудников:

Колонка Формат Обязательность
Номер Номер документа Обязательно
Дата выдачи ДД.ММ.ГГГГ Обязательно
Дата рождения ДД.ММ.ГГГГ Обязательно
Серия Серия документа Необязательно
ФИО Для отчёта Необязательно
1
Нажмите Открыть Госуслуги — откроется страница проверки РКЛ
2
Авторизуйтесь на Госуслугах (если ещё не вошли)
3
Вернитесь в расширение, перетащите Excel-файл в область загрузки
4
Проверьте предпросмотр данных и нажмите Начать проверку
5
Дождитесь завершения — результаты появятся в расширении
6
Нажмите Скачать отчёт (Excel) или Открыть полные результаты
Главный экран — перетащите Excel-файл в область загрузки
Предпросмотр загруженных данных на фоне страницы Госуслуг
Обновление
1
Скачайте новый архив rkl-check-extension.zip
2
Распакуйте в ту же папку (с заменой файлов)
3
Перейдите в chrome://extensions
4
Нажмите кнопку обновления (круглая стрелка) на карточке расширения РКЛ Check
Лицензия сохраняется при обновлении — повторная активация не требуется.
Настройки

Нажмите на шестерёнку в расширении для доступа к настройкам:

Задержка между проверками
Рекомендуется 10 секунд. При меньшем значении Госуслуги могут ограничить частоту запросов.
Авто-скачивание отчёта
Автоматически скачивать Excel-файл с результатами при завершении проверки.
Уведомления
Chrome-уведомление при завершении проверки всех сотрудников.
Экран настроек — задержка, авто-скачивание, уведомления
Ваши данные защищены
Данные сотрудников (ФИО, номера документов, даты рождения) не покидают ваш браузер. Сервер лицензий получает только лицензионный ключ и количество проверок.
